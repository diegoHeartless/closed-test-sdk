# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.2.x   | Yes       |
| < 0.2   | No        |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report security issues through [GitHub Security Advisories](https://github.com/diegoHeartless/closed-test-sdk/security/advisories/new) (preferred) or by contacting the maintainers via the email listed in the Maven POM / GitHub profile.

We aim to acknowledge reports within **3 business days** and to provide a fix or mitigation plan within **30 days** for confirmed issues, depending on severity.

## Scope

- SDK client code in this repository (`io.closedtest.sdk`)
- Published Maven artifacts `com.groundspaceteam:closed-test-sdk` and `com.groundspaceteam:closed-test-sdk-navigation-compose`

Out of scope: the Dozenflow / ingest backend, organizer apps, and host applications that embed the SDK.
